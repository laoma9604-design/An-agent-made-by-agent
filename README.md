# Herrs Desktop

> Hermes 的穷鬼表弟。DeepSeek 驱动，能动手不动嘴的桌面 AI。

## 这玩意是啥

**Hermes** 很牛逼——80 个工具、有记忆、跨平台、能自我进化。但它装起来烦。

**Herrs** 就是我把 Hermes 的皮扒了，留了最核心的东西：**AI 应该干活，不是光逼逼**。然后拿 Python 糊了个窗口。

| | Hermes | Herrs（就这） |
|------|------|------|
| 工具 | 80+ | 18 |
| 安装 | 一堆脚本 | 双击 一键安装.bat |
| 启动 | 开终端打 hermes | 双击 启动.bat |
| 关系 | 航母 | 橡皮艇 |

橡皮艇也能过河。

## 能干啥

| | 能干的事 |
|------|------|
|  文件 | 读、写、搜——比你翻文件夹快 |
|  系统 | 跑命令、跑 Python、杀进程、查磁盘 |
|  网络 | 搜索（Bing）、抓网页、下文件 |
|  桌面 | 打开东西、剪贴板、截图、弹通知 |

## 两种模式

| |  工具 |  纯聊 |
|------|------|------|
| 干嘛的 | 真动手干活 | 纯聊天 |
| 比如 | "帮我把C盘垃圾清了" | "给我讲讲什么叫反向传播" |
| Skill 面板 | 显示 | 隐藏 |
| 走代理 | ✅ 自动检测 | ✅ 自动检测 |

点顶部胶囊按钮切换，视觉差异明显，不会搞混。

## v5 新特性

**代理自动检测** — 不用手动填代理地址了。启动时自动扫 7890/7891/10809 等常见端口，扫到就自动配好，下次秒启。

**SSL 容错** — DeepSeek 间歇性 SSL 抽风自动重试，不等你手动点。

**进度可视化** — 状态栏会显示模型在干嘛：📡调用API → 🧠思考中 → 🔧执行工具 → ✍️生成回复。

## 跟一般 AI 有啥区别

你问 ChatGPT "C盘满了咋办" → 它写 500 字教程。

你问 Herrs → 它自己扫一遍，直接开删。

**不是它聪明，是它真敢动手。**

## 新电脑怎么装

```
1. 装 Python 3.10+（勾选 "Add Python to PATH"）
2. 双击 一键安装.bat
3. 启动后点右上角 🔑 设置 API Key
4. 代理不用管，自动检测（前提是你有代理在跑）
```

就这点事。完。

## 打包成 EXE

```bash
pip install pyinstaller
pyinstaller --onefile --noconsole --name "Herrs" main.py
```

exe 在 `dist/Herrs.exe`。

## 其他

- 80+ Skill 在左边（虽然目前只当提示词用）
- 关窗口缩托盘，`Ctrl+Shift+H` 叫回来
- 权限不足会自动提示（比如需要管理员运行）
- Key 存本地 `herrs_config.json`，不上传
- `my_skills/` 里放自己的 Skill
